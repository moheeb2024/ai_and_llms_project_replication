# Datasets Used in the Project

This folder contains all datasets required to replicate the analysis. The data processing pipeline proceeds as follows:

## 1. Raw Data

- **manifestos_48/**  
  Contains the original 48 German party manifesto CSV files.

## 2. Combined Dataset

- **combined.csv**  
  All manifesto files were merged into a single dataset.  
  This file contains the full corpus of quasi-sentences used in the study.

## 3. Migration Identification

- **df_final_migration_and_non-migration.csv**  
  Generated by applying Prompt 7 (see main text) to identify migration-related content.  
  This dataset includes both migration and non-migration quasi-sentences.

## 4. Filtered Migration Dataset

- **df_migration.csv**  
  Subset containing only migration-related quasi-sentences.  
  This dataset is used as input for the classification stage.

## 5. Manual Annotation (Training Data)

- **manually_coded_750_sample.csv**  
  A random sample of 750 quasi-sentences manually coded by human annotators.  
  This dataset is used to fine-tune the RoBERTa model.

## 6. Final Classified Dataset

- **df_migration_roberta_labeled.csv**  
  Output of the fine-tuned RoBERTa model.  
  Each quasi-sentence is assigned a frame label (humanitarian, economic, security, cultural).

## 7. External Datasets (Figures)

- **dataset_for_figure_1_eurostat.csv**  
  Eurostat data used for Figure 1 (see main text for citation)

- **dataset_for_figure_2_bamf.csv**  
  BAMF data used for Figure 2 (see main text for citation)
